# sample-bash-calculator
##Approach
1. Read Two Numbers
2. Input Choice (1-Addition, 2-Subtraction, 3-Multiplication, 4-Division) 
3. if Choice equals 1
    Calculate res = a + b
   else If Choice equals 2
    Calculate res = a - b
   else if Choice equals 3
    Calculate res = a * b
   else if Choice equals 4
    Calculate res = a / b
4. Output Result, res
